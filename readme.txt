Dependencies:

python (>= 2.6)
python-dev (= 2.6)
pyopengl (>= 3.0)
numpy (>= 1.4.1)

To make screenshots, you also need the Python Imaging Library (I have
  1.1.6-2 installed.).

To compile the library:

python setup.py build

To install the library:

sudo python setup.py install

Start using trep by checking out the examples in trep/examples/.
