Trep: Dynamic Simulation and Optimal Control¶

Release:0.91
Date:March 12, 2012

Trep is a Python module for modeling rigid body mechanical systems in generalized coordinates. Trep is designed to be fast, easy to use, and easy to extend. Trep currently features:

  • Concise System Definitions
  • Conservative Forces (Potential Energies)
  • Non-Conservative forces
  • Holonomic Constraints
  • Continuous Dynamics
  • Exact 1st and 2nd derivatives of continuous dynamics
  • Discrete Dynamics using Variational Integrators
  • Exact 1st and 2nd derivatives of discrete dynamics
  • Projection-based Trajectory Optimization in discrete time
  • Automatic Visualization Tools
  • Excellent Scalability

We’re still getting all of this documentation together. For now, check out the Installation page to get setup using trep and checkout the module reference.

If you have any questions or suggestions, please head over to our sourceforge page and contact us by email.
